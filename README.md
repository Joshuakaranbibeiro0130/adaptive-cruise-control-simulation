# Adaptive Cruise Control Simulation

This project demonstrates a simulation of an Adaptive Cruise Control (ACC) system using MATLAB and Simulink concepts.

## Overview
Adaptive Cruise Control is an advanced driver assistance system that automatically adjusts vehicle speed to maintain a safe distance from the vehicle ahead.

The system processes sensor inputs such as:
- Vehicle speed
- Distance to the leading vehicle
- Relative velocity

Based on this data, the controller dynamically adjusts throttle or braking to maintain a safe following distance.

## Features
- Dynamic speed adjustment
- Safe distance maintenance
- Sensor data processing
- Simulation of real-world driving scenarios

## Technologies
- MATLAB
- Simulink
- Control system logic

## Algorithm Logic
1. Measure distance from the vehicle ahead.
2. Compare with safe distance threshold.
3. Adjust speed accordingly.
4. Maintain smooth acceleration and braking.
